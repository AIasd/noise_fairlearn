# noise_fairlearn
## Introduction
This repo implements a general interface for noisy fair binary classification. It currently supports the following:

### fairness criteria:

* Demographic Parity
* Equality of Opportunity

### classifiers:

* Zafar
* Agarwal

### datasets:

 * UCI Adult
 * COMPAS
